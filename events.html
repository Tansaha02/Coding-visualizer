Chapter 05

Events & Listeners

Events are things the user does — click, type, hover. JavaScript can listen and respond.

01 — Common Events
Event Types
click
User clicks the element
mouseover
Mouse moves over element
keydown
A key is pressed
keyup
A key is released
submit
A form is submitted
input
Value changes in a field
load
Page finishes loading
dblclick
User double-clicks
const btn = document.getElementById("myBtn"); btn.addEventListener("click", function() { console.log("Button was clicked!"); });
