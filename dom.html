Chapter 04

DOM Manipulation

The DOM is the live structure of your webpage. JavaScript can select, change, add, and remove any element.

01 — Key DOM Methods
MethodWhat it doesExample
getElementById()Select one element by its iddocument.getElementById("title")
querySelector()Select first match by CSS selectordocument.querySelector(".card")
textContentGet or set the text inside an elementel.textContent = "Hello"
styleChange CSS directly from JSel.style.color = "red"
classList.add()Add a CSS class to an elementel.classList.add("active")
createElement()Create a brand new HTML elementdocument.createElement("p")
appendChild()Add a child element inside anotherparent.appendChild(newEl)

03 — The DOM Tree
HTML → DOM Tree
<html>
  ├─ <head>
  │  └─ <title> "CodeViz" </title>
  └─ <body>
     ├─ <h1 id="title"> "Hello" </h1>
     └─ <p class="info"> "World" </p>
Every HTML element becomes a node in this tree. JavaScript travels up and down to find and change any node.
